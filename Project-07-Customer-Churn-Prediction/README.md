#  Customer Churn Prediction

# Problem

Predict whether a customer is likely to leave (churn) or stay with the company based on customer data.

# Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Streamlit

# Analysis

Performed Exploratory Data Analysis (EDA) and statistical analysis to identify factors influencing customer churn such as tenure, monthly charges, and customer demographics.

# Features Used

* Tenure
* Monthly Charges
* Total Charges
* Senior Citizen
* Partner
* Dependents

# Model

* Classification model trained using Scikit-learn

# Deployment

Project-7 Customer_Churn_Predictor
https://huggingface.co/spaces/Sanjeet-ml/CUSTOMER_CHURN_PREDICTOR

# Output

* Churn Probability
* Customer likely to churn 
* Customer likely to stay
