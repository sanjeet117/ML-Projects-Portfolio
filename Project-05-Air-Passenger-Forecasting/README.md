#  Air Passenger Forecasting (SARIMA)

# Problem

Forecast future airline passenger traffic based on historical data using time series analysis.

# Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Statsmodels (SARIMA)
* Streamlit

# Analysis

Performed time series analysis including trend, seasonality, and stationarity checks. Applied log transformation to stabilize variance.

# Features Used

* Historical passenger count (time series)

# Model

* SARIMA (Seasonal ARIMA) model for forecasting
