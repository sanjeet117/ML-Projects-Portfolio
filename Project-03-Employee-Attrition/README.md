#  Employee Attrition Prediction

# Problem

Predict whether an employee is likely to leave the company or stay.

# Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

# Analysis

Performed detailed Exploratory Data Analysis (EDA) and statistical analysis to understand key factors affecting employee attrition.

# Features Used

* Age
* Monthly Income
* Job Satisfaction
* Years at Company
* OverTime

# Model

* Classification model trained using Scikit-learn

# Deployment

 Live Demo:
https://huggingface.co/spaces/Sanjeet-ml/employee-attrition-predictor

# Output

* Employee likely to leave 
* Employee likely to stay
