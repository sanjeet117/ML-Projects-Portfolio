# Sentiment Analysis NLP App

# Project Overview
This project is a Sentiment Analysis system that predicts whether a given text is **Positive** or **Negative**.

# Features
- Text preprocessing (stopwords removal, cleaning)
- TF-IDF vectorization
- Logistic Regression model
- Model comparison with Naive Bayes
- Deployed using Gradio

# Tech Stack
- Python
- Scikit-learn
- NLP
- Gradio

# Model Performance
- Accuracy: ~89%
- Balanced Precision & Recall
