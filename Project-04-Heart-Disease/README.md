# Heart Disease Prediction

 # Problem

Predict whether a person is likely to have heart disease based on medical attributes.

# Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

# Analysis

Performed Exploratory Data Analysis (EDA) and statistical analysis to understand patterns in patient data and identify important health indicators.

# Features Used

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Maximum Heart Rate
* Exercise Induced Angina

# Model

* Classification model trained using Scikit-learn
