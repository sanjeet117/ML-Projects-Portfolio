#  Customer Segmentation using KMeans

# Problem

Segment customers into different groups based on their behavior and spending patterns.

# Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

# Analysis

Performed Exploratory Data Analysis (EDA) to understand customer distribution and identify meaningful patterns in the data.

# Features Used

* Annual Income
* Spending Score
* Customer behavior features

# Model

* KMeans Clustering (Unsupervised Learning)

# Output
* Customer segments/groups
* Cluster visualization

# Note
This project focuses on clustering and data analysis. Deployment is not included.
