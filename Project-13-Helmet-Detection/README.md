#  Helmet Detection using YOLOv8

# Problem

Detect whether a person is wearing a helmet or not using object detection.

# Tech Stack

* Python
* YOLOv8 (Ultralytics)
* OpenCV
* NumPy
* Gradio

# Analysis

Trained a custom object detection model using YOLOv8 on helmet detection dataset.

# Features Used

* Image data (object detection bounding boxes)

# Model

* YOLOv8 custom trained model for object detection

# Deployment
Project-13
https://huggingface.co/spaces/Sanjeet-ml/yolo-helmet-detection

# Output
* Detected objects with bounding boxes
* Helmet / No Helmet detection
