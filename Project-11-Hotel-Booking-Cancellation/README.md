#  Hotel Booking Cancellation Prediction

# Problem

Predict whether a hotel booking will be canceled or not based on customer and booking details.

# Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn

# Analysis

Performed Exploratory Data Analysis (EDA) and statistical analysis to understand factors influencing booking cancellations.

# Features Used

* Lead Time
* Arrival Date (Year, Month, Week, Day)
* Number of Guests
* Previous Cancellations
* Booking Changes
* Waiting Time
* ADR (Average Daily Rate)
* Customer Type
* Market Segment
* Room Type

# Model

* Classification model trained using Scikit-learn

# Output

* Booking will be canceled 
* Booking will not be canceled 

# Note

This project focuses on model building and feature engineering. Deployment is not included.
