#  Walmart Weekly Sales Forecasting

# Problem

Predict future weekly sales for Walmart stores using historical sales data and time-based features.

# Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* Statsmodels
* Streamlit

# Analysis

Performed time series analysis and feature engineering using lag features, rolling statistics, and seasonal patterns to capture sales trends.

# Features Used

* Lag Features (Previous week sales)
* Seasonal Lag (Sales from same week last year)
* Month and Week number
* Rolling Mean and Rolling Standard Deviation

# Model

* Random Forest Regression model for sales forecasting
