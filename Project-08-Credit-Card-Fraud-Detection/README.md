#  Credit Card Fraud Detection

# Problem

Detect whether a credit card transaction is fraudulent or normal using machine learning.

# Tech Stack

* Python
* NumPy
* Scikit-learn
* XGBoost
* Streamlit

# Analysis

Performed Exploratory Data Analysis (EDA) on highly imbalanced transaction data. Applied preprocessing and scaling to handle numerical features effectively.

# Features Used

* Transaction Time
* Transaction Amount
* Engineered features (based on dataset structure)

# Model

* XGBoost classification model trained for fraud detection

# Deployment

Project-8 Credit_Card_Fraud_Detection

# Output
* Fraud Transaction 
* Normal Transaction
